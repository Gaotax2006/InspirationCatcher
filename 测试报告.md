## 三、已知注意事项

1. **AtlantaFX 兼容性**: AtlantaFX 2.0.1 基于 JavaFX 17+ 开发，与 JavaFX 21 理论上兼容。如果遇到控件样式异常，可以注释掉 `MainApp.java` 中的 `Application.setUserAgentStylesheet(new PrimerLight().getUserAgentStylesheet())` 回退到 JavaFX 默认样式
2. **CSS 层叠**: 自定义 CSS 在 AtlantaFX 主题之上加载，优先级更高。如果某些控件没有应用预期样式，可能需要增加 CSS 选择器的特异性
3. **ControlsFX**: 只是添加了依赖，尚未在代码中使用。后续可以用 `Notifications` 替换 `Alert`、用 `ToggleSwitch` 替换 CheckBox
4. **Ikonli**: 依赖已添加，但尚未替换所有 emoji。后续可通过 `FontIcon` 类替换按钮中的 emoji 文本
