# 灵感捕手 (Inspiration Catcher) — 设计报告

## 项目概述

灵感捕手是一款面向创作者和知识工作者的 JavaFX 桌面应用，专注于灵感的捕捉、整理、关联和可视化，集成 AI 辅助扩展功能。

## 架构设计

### 分层架构

```
Controller → Manager → DAO → SQLite
```

- **Controller**: 处理 FXML 事件和 UI 绑定 (MainController, EditorController, FilterController, AIPanelController, QuickCaptureController)
- **Manager**: 业务逻辑和状态管理 (IdeaManager, ProjectManager, MindMapManager, TableManager, TagManager, FontManager 等)
- **DAO**: 原始 JDBC 数据访问 (IdeaDao, TagDao, ProjectDao, MindMapNodeDao, MindMapConnectionDao)
- **Model**: 数据实体 (Idea, Project, Tag, MindMapNode, MindMapConnection, Connection)
- **Service**: 外部服务集成 (AIService — DeepSeek API 流式调用)
- **Component**: 自定义 UI 组件 (MindMapView — Canvas 思维导图渲染器)

## 核心功能

### 灵感管理 (CRUD)
- 快速捕捉（模态窗口）和完整编辑（Markdown 编辑器）
- 多维分类：项目、类型（7种）、心情（10种）、重要性（1-5）、标签
- 全文搜索 + 多条件复合过滤

### 思维导图
- Canvas 自定义渲染，支持缩放/平移
- 节点类型：灵感节点、概念节点、外部链接节点
- 拖拽创建连接、右键菜单编辑
- 连接类型：RELATED/DEPENDS_ON/EXTENDS/CONTRADICTS/ANALOGY/CAUSAL/USES
- 从侧边栏拖拽灵感卡片到画布

### AI 辅助
- 基于 DeepSeek API 的流式响应 (SSE)
- 两种模式：快速建议 + 自定义提示词扩展
- Jackson JSON 解析，HttpClient 异步调用

### 主题
- AtlantaFX PrimerLight 为基础，自定义 Warm Paper 暖色覆盖
- 主色 `#C4843C`（陶土色），背景 `#F7F4F0`（暖米白）
- CSS 变量体系：设计令牌、组件样式、工具类
